# My Python Guessing Game 

This is my final project for my Python coding class. It’s a fun, interactive game where the computer picks a random number and you have to guess what it is.

## What it does
*   **Pick a Number: The computer chooses a secret number between 1 and 10.
*   **Gives Hints: It tells you if your guess is "too high" or "too low" so you can try again.
*   **Tracks Progress:** It counts how many tries it takes you to win.

## ✨ What I learned
While making this, I practiced using:
*   **Random library** for the secret number.
*   **While loops** to keep the game running until you win.
*   **If/Else statements** to check the user's guesses.
